### logging库 
- basicConfig配置日志打印（p82）
```python
import logging
logging.basicConfig(level=logging.INFO,                               #设置日志级别为 INFO
                    format='%(asctime)s - %(levelname)s: %(message)s')#设置打印格式: 时间 - 级别:日志消息
```
